# 兄妹合作闯关之旅 (Nice Try) - 详细设计与启动文档

## 🎮 游戏定位与初衷
这是一款专门为 **8岁哥哥** 和 **3岁妹妹** 设计的**非对称双人合作**本地网页小游戏合集。
旨在通过游戏化的机制，潜移默化地引导孩子们学会：
- **礼貌地请求**（不硬抢、说“请”）
- **友好沟通与商量**（共同决策才能前进）
- **按规矩排队/轮流**（学会等待，各司其职）

游戏适合在平板电脑（如 iPad）上平放游玩。哥哥坐一边，妹妹坐另一边，通过上下分屏进行互动。

---

## 🚀 游戏启动与运行说明 (家长必读)

本项目纯前端实现，无需安装复杂的数据库或依赖，只需通过 Python 启动一个简单的本地 HTTP 服务器即可在家庭局域网内游玩。

### 第一步：在电脑上启动游戏服务器
1. 打开 Mac/PC 的终端 (Terminal)。
2. 使用 `cd` 命令进入到本项目的 `nice-try` 目录：
   ```bash
   cd /Users/bytedance/Documents/git_repo/Vibe-coding/nice-try
   ```
3. 运行以下 Python 命令启动本地服务器（通常 Mac 和 Linux 系统都自带 Python3）：
   ```bash
   python3 -m http.server 8000
   ```
   *(如果看到类似 `Serving HTTP on :: port 8000 ...` 的提示，说明启动成功。)*

### 第二步：在 iPad 上访问游戏
为了获得最佳的平放对战体验，强烈建议使用 iPad 或大屏平板游玩。
1. 确保您的电脑和 iPad 连接在**同一个家庭 Wi-Fi 网络**下。
2. 查找您电脑的局域网 IP 地址：
   - Mac：按住 `Option` 键点击顶部菜单栏的 Wi-Fi 图标，查看“IP 地址”（如 `192.168.1.10`）。或者在终端输入 `ifconfig`。
   - Windows：在命令提示符中输入 `ipconfig`，查看“IPv4 地址”。
3. 在 iPad 打开 Safari 或 Chrome 浏览器，在地址栏输入：
   ```text
   http://您的电脑IP地址:8000
   ```
   例如：`http://192.168.1.10:8000`
4. 页面加载出来后，就可以把 iPad 平放在桌子中间，让孩子们开始玩啦！

*(注：如果需要清除游戏进度重新开始，可以直接点击主菜单底部的“重新开始游戏”按钮。)*

## 🗺️ 游戏关卡与核心机制说明

游戏包含一个主菜单和三个逐步解锁的关卡。只有通过了前一关，才能解锁下一关（进度保存在浏览器 `localStorage` 中）。

### 1️⃣ 第 1 关：魔法点心屋 (game_a.html)
- **学习目标**：礼貌请求（实体化“说请”）
- **哥哥（仓库管理员）**：负责看懂顾客订单，提供正确的食材。
- **妹妹（魔法点心师）**：看着搅拌碗提示向哥哥要食材。
- **防抢夺机制**：如果妹妹直接去点哥哥的食材，食材会发出“啵”的拒绝声并原地抖动。妹妹必须按下大大的“✨ 请给我”按钮，播放可爱的音效后，哥哥那边的食材才会解锁。
- **通关条件**：成功合作制作两份点心。

### 2️⃣ 第 2 关：双人寻宝船 (game_b.html)
- **学习目标**：友好商量与同步合作
- **玩法**：两人共同驾驶一艘小船。遇到岔路口时，哥哥和妹妹的屏幕会出现相同的两个选项（如苹果/香蕉）。
- **商量机制**：两人**必须大声商量，并点击选择同一个图案**，小船才会前进。如果各选各的，小船会在原地打转晕头转向。
- **通关条件**：连续 5 次达成一致选择，找到宝藏。

### 3️⃣ 第 3 关：动物救援桥 (game_c.html)
- **学习目标**：各司其职与轮流等待
- **玩法**：小动物需要过河。哥哥负责“放木板”，妹妹负责“钉钉子”。
- **轮流机制（红绿灯）**：引入了交通灯概念。绿灯亮时才能操作。如果轮到哥哥放木板时，妹妹抢着按“钉钉子”，或者反之，桥就会剧烈摇晃，提示“还没到你的回合”。强制孩子们学会等待对方完成操作。
- **通关条件**：按顺序成功铺设并钉好 5 块木板，让小动物顺利过河。

## 🎨 UI 与技术栈
- **技术栈**：原生 HTML5 + CSS3 + 原生 JavaScript。无需打包编译工具。
- **UI 风格**：
  - **儿童友好**：采用马卡龙/糖果色渐变背景，主菜单使用清爽的浅蓝色调。
  - **专属定制**：标题专门定制为“👦👧 谦谦&闲闲 大闯关～”，增强孩子们的代入感。
  - **Q弹交互**：按钮和卡片采用粗边框加大圆角设计，底部带有粗阴影。点击时有明显的“下沉”物理按压反馈。
  - **动效与音效**：使用 CSS 关键帧动画（悬浮、呼吸发光、错误抖动）。
- **纯代码生成的背景音乐 (Web Audio API)**：
  为了保持项目轻量且无需下载外部 MP3 文件，游戏使用浏览器原生的 Web Audio API 编写了 4 首风格各异的儿歌背景音乐（每个页面右上角均有 🎵 控制开关）：
  - **主菜单**：《一闪一闪小星星》（柔和正弦波，类似八音盒质感）
  - **第 1 关 (魔法点心屋)**：《我是一个粉刷匠》（欢快三角波，配合做蛋糕的劳动氛围）
  - **第 2 关 (双人寻宝船)**：《划船歌 Row Row Row Your Boat》（水波般柔和的正弦波）
  - **第 3 关 (动物救援桥)**：《两只老虎》（活泼方波，带点复古电子游戏风，配合小动物过河）
